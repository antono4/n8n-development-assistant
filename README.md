# Free LLM Chat - n8n AI Assistant

> **Created by Antono**


Chat with AI using completely **FREE** LLM APIs. No API keys required for most providers!

## ⚡ Features

- **5 Free LLM Providers** - Groq, OpenRouter, HuggingFace, GitHub Models, Cloudflare
- **No API Keys Required** - Enter keys once, they're saved locally
- **Multiple Models** - Llama 3, Mixtral, Qwen, Mistral, Gemma and more
- **n8n Focused** - Specialized for workflow automation and development
- **Dark Theme** - Beautiful UI matching n8n's style

## 🚀 Quick Start

```bash
# Open in browser
file:///path/to/index.html

# Or use a local server
python3 -m http.server 8080
# Then open http://localhost:8080
```

## 🔑 Get API Keys

1. **Groq** - https://console.groq.com/keys (Free, no credit card)
2. **OpenRouter** - https://openrouter.ai/keys (Free credits)
3. **HuggingFace** - https://huggingface.co/settings/tokens (Free tier)
4. **GitHub Models** - https://github.com/settings/tokens (Free for Copilot)
5. **Cloudflare** - https://dash.cloudflare.com/profile/api-tokens (Free tier)

## 💡 Usage

1. **Select a Provider** - Click on a provider in the sidebar
2. **Enter API Key** - First time will prompt for key
3. **Choose Model** - Select from available models
4. **Start Chatting** - Ask anything about n8n, code, workflows!
